# ColorPicker 控件需求文档

> 本文档定义 UICornerstone 新增控件 ColorPicker 的需求，输入到 UICornerstone 项目进行独立实现。

## 1. 背景与动机

CornerstoneDesigner 的属性编辑器（F3.3）需要让用户选择颜色（如 Button 的 background color、Label 的 font color 等）。目前 UICornerstone 无颜色选择能力。

**需求来源**：CornerstoneDesigner F3.3。可用于任何需要颜色配置的场景。

## 2. 功能需求

| 编号 | 需求 | 优先级 | 说明 |
|------|------|--------|------|
| F1 | 颜色色块预览 | P0 | 显示当前颜色的矩形色块，点击触发弹窗 |
| F2 | 调色板弹窗 | P0 | 点击色块弹出颜色选择窗口 |
| F3 | 预设颜色面板 | P0 | 提供 20+ 常用预设色（红/绿/蓝/灰/白/黑 等），单击直接选中 |
| F4 | 十六进制输入 | P0 | 在弹窗中直接输入 #RRGGBB 文本 |
| F5 | RGB 滑块 | P1 | R/G/B 各一个滑块（0-255），实时预览混合色 |
| F6 | 取色器（吸管） | P2 | 拾取屏幕上任意位置的颜色 |
| F7 | 不透明度 Alpha | P2 | 支持 #RRGGBBAA 格式 |

## 3. 数据结构

```cpp
class ColorPicker : public ControlImpl {
public:
    // 颜色值
    void setColor(const std::string& hex);  // #RRGGBB
    std::string getColor() const;

    // 事件
    void setOnColorChangedCallback(
        std::function<void(const std::string& hex)> callback);
};
```

## 4. 事件

| 事件 | 触发时机 | 回调参数 |
|------|---------|---------|
| onColorChanged | 颜色改变时 | 新的 #RRGGBB 字符串 |

## 5. JSON 布局格式

```json
{
  "type": "ColorPicker",
  "id": "btnBgColor",
  "rect": { "x": 10, "y": 10, "w": 60, "h": 24 },
  "color": "#4A90D9",
  "events": { "onColorChanged": "onBgColorChanged" }
}
```

## 6. 渲染规范

### 6.1 闭合状态

```
┌──────────┐
│ ■ #4A90D9│
└──────────┘
```

- 左侧：16x16 像素色块，显示当前颜色
- 右侧：十六进制颜色文本

### 6.2 弹窗状态

```
┌──────────────────────┐
│  预设色面板 (5x5 网格) │
│  [■][■][■][■][■]    │
│  [■][■][■][■][■]    │
│  [■][■][■][■][■]    │
│  [■][■][■][■][■]    │
│  [■][■][■][■][■]    │
│                      │
│  # [4A90D9        ]  │
│                      │
│  R [=====●=====] 145 │
│  G [==●=========] 72  │
│  B [========●===] 217 │
│                      │
│  [确定]  [取消]        │
└──────────────────────┘
```

- 弹窗为浮于其他控件之上的独立 Panel
- 预设色块点击即选中，弹窗不关闭
- 十六进制输入框支持直接键盘编辑
- RGB 滑块拖动实时更新色块预览
- 确定/取消按钮点击后关闭弹窗

### 6.3 预设色列表

```
#FF0000  #00FF00  #0000FF  #FFFF00  #FF00FF
#00FFFF  #FFFFFF  #000000  #808080  #C0C0C0
#800000  #808000  #008000  #800080  #008080
#000080  #FFA500  #FFC0CB  #A52A2A  #F0F8FF
```

## 7. 边界与约束

- 当前版本仅支持 #RRGGBB（6 位十六进制），暂不支持 #RGB 缩写
- 颜色值校验：非法字符忽略，不足 6 位左补 0

## 本阶段测试内容

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 色块预览 | 设 #FF0000 | 色块显示红色 |
| 预设选择 | 点击预设色 | 选中的颜色正确 |
| 十六进制输入 | 输入 #3366CC | 色块变为蓝色 |
| RGB 滑块 | 拖动 R 滑块 | 红色分量变化，色块同步 |
| 事件 | 颜色变化 | onColorChanged 触发 |
| JSON | 全参数/缺省 | 解析正确 |
| 弹窗关闭位置 | 打开 -> 确定 -> 重新打开 | 弹窗位置合理 |
